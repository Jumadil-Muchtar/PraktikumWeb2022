Selamat Datang

Ini adalah blog yang saya buat untuk memenuhi tugas praktikum mata kuliah Pemrograman Web. Halaman web ini disusun hanya menggunakan html saja dan tidak menggunakan CSS ataupun framework CSS seperti Bootstrap dan lainnya.
Tugas praktikum yang saya buat adalah sebuah blog yang terdiri dari halaman Home, Education, dan About.
Halaman Home berisi tentang pengenalan blog serta tujuan blog dibuat, halaman Education berisi tentang riwayat pendidikan praktikan, dan halaman About berisi tentang biodata praktikan.
